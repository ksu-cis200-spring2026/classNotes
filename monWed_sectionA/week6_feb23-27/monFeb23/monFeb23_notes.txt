Strings. 

String first = "hello";

How to print all the characters in first, one per line?

for (int i = 0; i < first.length(); i++) {
    System.out.println(first.charAt(i));
}






What is this doing?

String first = "hello";
String second = "goodbye";
for (int i = 0; i < second.length(); i++) {
    first += second.charAt(i);
}

first will be: ? hellogoodbye



...is there a more efficient way? StringBuilder

String first = "hello";
String second = "goodbye";
StringBuilder build = new StringBuilder(first);
for (int i = 0; i < second.length(); i++) {
    build.append(second.charAt(i));
}

String split.

String big = "This is a sentence.";


--> How to print the individual words?

String[] pieces = big.split(" ");
for (int i = 0; i < pieces.length; i++) {
    System.out.println(pieces[i]);
}



--> For-each loops

String[] pieces = big.split(" ");
for (String elem : pieces) {
    System.out.println(elem);
    elem = "test"; //won't compile!
}




--> What if there are multiple spaces in a row?

String big = "This    is a sentence.";

String[] pieces = big.split("\\s+");
for (int i = 0; i < pieces.length; i++) {
    System.out.println(pieces[i]);
}
