In Vector3:

- magnitude (already written)
    - takes a vector (array of doubles)
    - returns the length of the vector (a double)
- addVectors
    - takes two vectors (as arrays of doubles)
    - add/returns (new) sum of parameter vectors
- printVector
    - takes a vector (array of doubles)
    - prints it in the format "(1.23, 4.56, 7.89)"
- normalize
    - takes a vector (as an array of doubles)
    - MODIFIES the parameter vector to be normalized
    (divide each parameter by the length)
    (need to SAVE the original length before changing the vector)
