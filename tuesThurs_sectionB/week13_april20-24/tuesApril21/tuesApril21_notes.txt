keys: must be unique



Want keys to be names, values to be Person objects

HashMap<String, Person> map = new HashMap<String, Person>();

//Bob, 20
map.put("Bob", new Person("Bob", 20));
map.put("Alice", new Person("Alice", 30));
map.put("Fred", new Person("Fred", 10));
map.put("Bob", new Person("Bob", 25)); //replaces the old Bob


--> add a few people 
--> how would we find the oldest one?

int bigAge = 0;
Person oldestPerson = null;
for (String name : map.keySet()) {
    Person cur = map.get(name);
    if (cur.getAge() > bigAge) {
        bigAge = cur.getAge();
        oldestPerson = cur;
    }
}
System.out.print(oldestPerson);
